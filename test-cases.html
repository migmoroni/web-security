🔒 Web Security Analyzer - Casos de Teste

Esta página contém links para testar o sistema de análise dinâmica da extensão Web Security Analyzer.

📍 Casos de Teste - Alfabetos Mistos

🚨 Cyrillic + Latin (Muito Suspeito)

gооglе.com
⚠️ Contém caracteres cirílicos 'о' no lugar de 'o' latinos. Análise esperada: Detecção de homóglifos cyrillic/latin + similaridade alta com google.com

🚨 Greek + Latin (Phishing Sofisticado)

αpple.com
⚠️ Primeira letra é alpha grego (α) em vez de 'a' latino. Análise esperada: Scripts mistos + similaridade com apple.com

🚨 Arabic + Latin (Mistura Complexa)

faϲebook.com
⚠️ Caractere 'ϲ' (Greek lunate sigma) no lugar de 'c'. Análise esperada: Alfabetos mistos + alta similaridade com facebook.com

🎯 Casos de Teste - Similaridade de Domínio

🚨 Typosquatting (Troca de Letras)

goggle.com
⚠️ 'goggle' em vez de 'google' - uma letra 'o' extra. Análise esperada: Alta similaridade com google.com + diferenças específicas

🚨 Subdomain Spoofing

paypal-security.com
⚠️ Usa 'paypal' como parte do domínio para enganar usuários. Análise esperada: Imitação de domínio + phishing detection

🚨 Character Substitution

arnazon.com
⚠️ 'rn' no lugar de 'm' em amazon.com. Análise esperada: Alta similaridade + análise de diferenças específicas

✅ Casos de Teste - Sites Legítimos com Subdomínios

✅ Google com Subdomínio

accounts.google.com/signin
✅ Subdomínio legítimo do Google. Análise esperada: Domínio extraído = "google.com" → Nenhuma ameaça

✅ Amazon com Path Complexo

amazon.com/dp/B08N5WRWNW/ref=sr_1_1
✅ URL complexa da Amazon com path longo. Análise esperada: Domínio extraído = "amazon.com" → Nenhuma ameaça

✅ Microsoft com Subdomínio e Path

login.microsoftonline.com/common/oauth2/authorize
✅ Subdomínio legítimo da Microsoft. Análise esperada: Domínio extraído = "microsoftonline.com" → Nenhuma ameaça

✅ Site Legítimo

google.com
✅ Site completamente legítimo. Análise esperada: Nenhuma ameaça detectada

✅ Site Brasileiro Legítimo

uol.com.br
✅ Portal brasileiro conhecido. Análise esperada: Nenhuma ameaça detectada

🔬 Como Testar

  1. Instale a extensão Web Security Analyzer
  2. Clique nos links suspeitos acima
  3. Observe os avisos de segurança da extensão
  4. Compare com os resultados esperados

🛡️ Recursos Testados

  • Extração de Domínio Inteligente: Analisa apenas o domínio principal, ignorando subdomínios e paths
  • Análise de Alfabetos Mistos: Detecta mistura de scripts (Latin, Cyrillic, Greek, Arabic, etc.)
  • Detecção de Homóglifos: Identifica caracteres que imitam letras latinas
  • Similaridade de Domínio: Compara com base de 130+ sites legítimos
  • Análise de Diferenças: Mostra exatamente quais caracteres diferem
  • Whitelist Inteligente: Não alerta para subdomínios de sites conhecidos
  • Scoring Inteligente: Calcula nível de ameaça baseado em múltiplos fatores

🔍 Exemplos de Extração de Domínio

  • accounts.google.com/signin/oauth → google.com
  • www.mail.yahoo.com/inbox → yahoo.com
  • subdomain.site.com.br/page → site.com.br
  • login.microsoftonline.com/common/oauth2 → microsoftonline.com
